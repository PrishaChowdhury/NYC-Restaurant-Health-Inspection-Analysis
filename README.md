# NYC Restaurant Health Inspection Analysis
Statistical analysis of 292K+ inspections using R.

## Key Findings
- 77.6% violation recurrence rate (McNemar's test, χ²=152.57, p<0.001)
- Strong correlation (r=0.67) between grade and violations
- 87% accuracy in grade prediction using logistic regression

## Methods
- Pearson correlation
- McNemar's test
- Logistic regression
- Data visualization (ggplot2)

## Files
- `analysis.R` - Main analysis
- `report.pdf` - Full findings
